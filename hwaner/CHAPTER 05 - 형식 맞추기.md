# 3. Java Class Declarations
- 오라클 자바 문서

    - public -> protected -> package -> private
  2. instance 변수
    - public -> protected -> package -> private
  3. 생성자
  4. 메서드
     - public 메서드에서 호출되는 private 메서드는 그 아래에 둔다. 가독성 위주로 그룹ing
